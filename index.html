The One Page Terminal Manual

A summary of useful terminal commands

System

man command
Show manual for command 'command'
shutdown -h now
Shutdown the system now and don't reboot
shutdown -r now reboot
Shutdown the system and reboot
shutdown -r 5
Shutdown the system in 5 minutes and reboot
halt
Stop all process
Ctrl + c
Halts current command
Ctrl + z
Stops current command
date
Print the current date/time
cal
Show current month calendar
uptime
Show computer uptime
w
Show who is online
whoami
Show who are you logged in as
uname -a
Show kernel config
cat /proc/cpuinfo
Cpu info
cat /proc/meminfo
Memory info
df
Show disk usage
du
Show directory space usage
du -h
Human readable size in GB
free
Show memory and swap usage
!!
Repeat las command
exit
Log out of current session

Files

ls -l
List files using long format
ls -al
Formatted listing with hidden files
ls -F
List files indicating the file type
ls -laC
List all files in long format and display in columns
cd dir
Change directory to dir
cd
Change to home
pwd
Print working directory
mkdir
Create directory dir
rm file
Remove file
rm -r dir
Remove directory dir
rm -f file
Force remove file
cp file1 file2
Copy file1 to file2
mv file1 file2
Rename file1 to file2
ln -s file link
Create symbolic link 'link' to file
touch file
Create or update file
cat > file
Place standard input into file
less file
Output the contents of the file
head file
Output first 10 lines of file
tail file
Output last 10 lines of file
tail -f file
Output contents of file as it grows

Ssh

ssh user@host
Connect to host as user
ssh -p port user@host
Connect to host as user using port -p

Network

ping host
Ping host 'host'
whois domain
Get whois for domain
dig domain
Get DNS for domain
wget file
Download file
wget -r url
Download recursively download files from url

Searching

grep pattern files
Search for pattern in files
grep -r pattern dir
Search recursively for pattern in dir
locate file
Find all instances of file

Process management

ps
Display currently active processes
ps aux
ps with a lot of deteils
kill pid
Kill process with pid 'pid'
killall proc
Kill all processes named proc
bg
Lists stopped/background jobs, resume stopped job in the background
fg
Bring most recent job to foreground
fg n
Bring job n to foreground

File permission

chmod octal file
Change permission of file
  • (4) read (r)
  • (2) write (w)
  • (1) execute (x)

Order: owner/group/world

chmod 777 - rwx for everyone

chmod 755 - rwx for owner, rx for group/world

Compression

tar cf file.tar files
Tar files into file.tar
tar xf file.tar
Untar into current directory
tar tf file.tar
Show content of archive

Tar flags:

  • c - create archive
  • t - table of contents
  • x - extract
  • f - for filename
  • z - use zip/gzip
  • j - bzip2 compression
  • k - do not overwrite
  • T - file from files
  • w - ask for confirmation
  • v - verbose
gzip file
Compress file and rename to file.gz
gzip -d file.gz
decompress file.gz
